Starling Bank Round-Up Service
Приложение для автоматизации накоплений, интегрированное с Starling Bank API (Sandbox). Проект анализирует транзакции пользователя и рассчитывает сумму «сдачи» для последующего перевода в накопительную корзину.

Основные возможности
Интеграция с REST API: Полноценная работа с банковскими эндпоинтами для получения счетов и истории транзакций.

Автоматизация накоплений: Расчет суммы округления (Round-Up) для всех исходящих транзакций за указанный период.

Безопасность: Реализация авторизации через Bearer-токен и кастомные HTTP-заголовки.

Надежность: Строгая фильтрация транзакций по статусу SETTLED и направлению OUT.

Архитектурные решения
Проект следует принципам Separation of Concerns и разделен на логические слои:

Networking Layer: Использование URLComponents и перечислений (StarlingEndpoint) для безопасного формирования запросов.

Data Layer: Модели данных используют протокол Decodable для маппинга JSON-ответов от API.

Business Logic: Вынос математических расчетов в отдельный сервис RoundUpService, что упрощает тестирование и поддержку.

Modern Concurrency: Использование Swift async/await вместо старых completion handlers для чистого и читаемого асинхронного кода.

Логика расчета (Round-Up)
Алгоритм работает по принципу «сдачи» до ближайшего целого фунта:

Берется остаток в пенсах от транзакции (amount % 100).

Если остаток больше нуля, вычисляется разница до 100 (100 - остаток).

Транзакции с нулевым остатком (£10.00, £5.00) игнорируются (сдача 0p).

Тестирование
Проект включает в себя Unit-тесты на фреймворке XCTest, которые покрывают ключевую финансовую логику:

Валидация расчета сдачи для разных сумм.

Проверка корректной обработки транзакций без остатка.

Гарантия того, что логика не учитывает входящие переводы.

Системные требования
iOS: 15.0+

Xcode: 15.0+

Swift: 5.9+

Как запустить
Склонируйте репозиторий.

Получите персональный токен в консоли разработчика Starling.

Вставьте его в CustomerDetails.swift в поле accessToken.

Запустите тесты нажатием Cmd + U, чтобы убедиться в стабильности сборки.
